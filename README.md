# Doceria Online Java

Projeto de uma loja virtual de doces desenvolvido em Java.

## Funcionalidades

- Listagem de produtos
- Adicionar ao carrinho
- Remover produtos
- Controle de quantidade
- Cálculo do valor total

## Tecnologias

- Java
- HttpServer
- HTML/CSS

## Como executar

1. Execute o projeto
2. Acesse:

http://localhost:8080
